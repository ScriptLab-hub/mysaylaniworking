Question 13: Create a new array. Store values one by one in such a way that you can access the values in the order in which they were stored. (FIFO - First In First Out)

This demonstrates a Queue data structure using JavaScript arrays. Elements are added to the end (`push()`) and removed from the beginning (`shift()`).

        
            let fifoArray = []; // Our FIFO queue

            // Adding elements (enqueue)
            fifoArray.push("Data 1");
            fifoArray.push("Data 2");
            fifoArray.push("Data 3");

            // Accessing/Removing elements (dequeue)
            let firstElement = fifoArray.shift(); // Removes "Data 1"
            let secondElement = fifoArray.shift(); // Removes "Data 2"
