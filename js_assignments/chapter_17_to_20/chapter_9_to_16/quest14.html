Question 14: Create a new array. Store values one by one in such a way that you can access the values in reverse order. (Last In - First Out)

This demonstrates a Stack data structure using JavaScript arrays. Elements are added to the end (`push()`) and removed from the end (`pop()`).

        
            let lifoArray = []; // Our LIFO stack

            // Adding elements (push)
            lifoArray.push("Book 1");
            lifoArray.push("Book 2");
            lifoArray.push("Book 3");

            // Accessing/Removing elements (pop)
            let lastInElement = lifoArray.pop(); // Removes "Book 3"
            let secondLastInElement = lifoArray.pop(); // Removes "Book 2"
