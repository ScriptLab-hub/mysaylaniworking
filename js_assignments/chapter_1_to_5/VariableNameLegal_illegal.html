Rules for naming JS variables

  1. Declare 3 variables in one statement:
    var a, b, c;
  2. 5 Legal variable names:
    • myVariable
    • _myVar
    • $dollar
    • name1
    • user_name
    5 Illegal variable names:
    • 1name (cannot start with a digit)
    • my-variable (hyphens not allowed)
    • var (reserved keyword)
    • user name (spaces not allowed)
    • @value (special characters not allowed)
  3. Rules:
    • Variable names can only contain letters, numbers, $ and _.
      For example: $my_1stVariable
    • Variables must begin with a letter, $ or _.
      For example: $name, _name or name
    • Variable names are case sensitive.
    • Variable names should not be JS keywords.
